# dotfiles

## Install

From Source:
```bash
sh -c "$(curl -fsLS https://chezmoi.io/get)" -- init --apply git.dayl.in/daylin -S ~/.dotfiles --ssh
```

From Mirror:
```bash
sh -c "$(curl -fsLS https://chezmoi.io/get)" -- init --apply daylinmorgan -S ~/.dotfiles --ssh
```
